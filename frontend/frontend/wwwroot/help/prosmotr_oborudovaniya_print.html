4.2 Просмотр оборудования

Для того, чтобы посмотреть имеющееся оборудование в системе необходимо с главного экрана перейти в раздел "Оборудование". В предыдущем разделе было описано, как это сделать. Теперь рассмотрим элементы, которые находятся в этом разделе.

Кнопка "Создать новый"

С помощью этой кнопки вы можете создать новый вид оборудования

Поиск

Это поле позволит вам найти необходимый вид оборудования по названию

Наименование

В данной колонке отображаются наименования оборудования

Количество

В этой колонке вы можете увидеть количество оборудования на складе и через дробь общее количество

Кнопка "Перейти"

Здесь вы можете перейти в оборудование для выполнения различных операций с ним
