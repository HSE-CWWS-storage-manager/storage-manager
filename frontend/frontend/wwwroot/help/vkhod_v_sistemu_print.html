3.1.2 Вход в Систему

Переходите к этому шагу после полного завершения регистрации. По окончании этого этапа вы попадёте на главную страницу снова. Рассмотрим ваши дальнейшие действия по шагам:
  • На главном экране Системы нажмите на кнопку "Войти"

    Кнопка "Войти"

    Необходимо нажимать сюда
  • Теперь необходимо ввести свой адрес электронной почты и пароль, указанный при регистрации, и нажать на кнопку "Войти"

    Поле ввода адреса электронной почты

    Введите сюда адрес электронной почты, указанный при регистрации

    Поле для ввода пароля

    Введите сюда пароль, указанный при регистрации

    Кнопка "Войти"

    Нажмите сюда после ввода всех данных
