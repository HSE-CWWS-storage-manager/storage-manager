2.1 Виды деятельности, функции

Система предназначена для автоматизации следующих функций:
  • функция ведения учета количества оборудования на складе;
  • функция ведения реестра передач оборудования студентам;
  • функция оформления списания оборудования;
  • функция формирования маркировки для оборудования;
  • функция формирования отчета о количестве оборудования на складе.
