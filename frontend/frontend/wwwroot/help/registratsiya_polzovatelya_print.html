3.1.1 Регистрация пользователя

Для обеспечения доступа к Системе каждый пользователь должен пройти регистрацию. Она состоит из двух этапов:
  • Регистрация в системе с помощью данных электронной почты и пароля;
  • Ожидание выдачи роли "Оператор" от администратора.
    Рассмотрим первый этап по шагам:
  • Необходимо перейти на главную страницы Системы по адресу, который вам сообщит администратор.

    Адресная строка браузера

    Сюда необходимо ввести адрес Системы, полученный от администратора
  • Теперь необходимо нажать на кнопку "Зарегистрироваться"

    Кнопка регистрации

    Необходимо нажать сюда
     
  • Теперь необходимо ввести адрес электронной почты, пароль и подтверждение пароля.

    Поле для адреса электронной почты

    Сюда необходимо ввести адрес электронной почты

    Поле для ввода пароля

    Сюда необходимо ввести свой пароль

    Поле для подтверждения пароля

    Введите сюда свой пароль снова

    Кнопка для завершения регистрации

    Нажмите сюда на этом ваша регистрация будет окончена
     
  • Дождитесь подтверждения вашей регистрации от администратора, после этого можно будет приступить к работе в Системе.
