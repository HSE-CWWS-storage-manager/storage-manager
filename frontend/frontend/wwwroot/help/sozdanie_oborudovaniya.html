4.1 Создание оборудования

Рассмотрим по шагам действия по созданию нового вида оборудования:
  • На главном экране перейдите в раздел "Оборудование"

    Кнопка перехода в раздел "Оборудование"

    Нажмите сюда, чтобы перейти в раздел с оборудованием
    Наверх
  • Нажмите на кнопку "Создать новый"

    Кнопка "Создать новый"

    Нажмите сюда для перехода в окно создания оборудования
    Наверх
  • Заполните необходимые данные об оборудовании и нажмите создать

    Наименование оборудования

    Заполните в этом поле наименование оборудования
    Наверх

    Количество на складе

    Введите количество оборудования, которое сейчас на складе
    Наверх

    Модель оборудования

    Введите в этом поле название модели оборудования
    Наверх

    Серийный номер

    Введите в это поле серийный номер оборудования
    Наверх

    Инвентарный номер

    Введите в это поле инвентарный номер оборудования
    Наверх

    Кнопка "Создать"

    После ввода всех данных нажмите на эту кнопку
    Наверх
